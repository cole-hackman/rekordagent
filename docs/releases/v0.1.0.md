# v0.1.0 Release Notes

> Draft until the real-library manual checklist passes and the tag is created.

## Summary

Foundation release for read-only Rekordbox 7 library browsing.

## Included

- Read-only SQLCipher access to Rekordbox `master.db`.
- Track table with filter/sort.
- Track detail panel with metadata and cues.
- Native audio preview play/pause.
- Settings for theme, library path, and Anthropic API key.
- Basic Claude chat with read-only search, playlist listing, and orphan scan tools.
- Rekordbox XML parse/emit crate.

## Known Limitations

- Waveform rendering is a placeholder.
- Scrub controls are deferred.
- Playlist UI is not dedicated yet.
- Chat conversations are not persisted.
- Agent cannot stage, review, or export changes yet.
