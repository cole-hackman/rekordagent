# v0.1.0 Release Notes

> Ready for manual library verification and final v0.1.0 tagging.

## Summary

Foundation MVP release for read-only Rekordbox 7 library browsing, intelligent agent auditing, and XML metadata export.

## Included

- Read-only SQLCipher access to Rekordbox `master.db`.
- Track table with filter/sort and "Global Library" layout.
- Track detail panel with clean metadata displays and hot cues.
- Native audio preview play/pause with placeholder visual representation.
- Settings for theme, library path, and Anthropic API key.
- Claude chat interface with intelligent orchestration to propose changes safely.
- Agent read-only MVP tools: search, get track, list playlists, get playlist, list cues, orphan scan, duplicate scan, broken metadata scan.
- Conversation persistence in local SQLite WAL cache.
- Safe staged changes system (`Proposed`, `Accepted`, `Rejected`, `Exported`).
- Grouped inline diff review UI with interactive status filters.
- Export accepted changes to Rekordbox XML with full support for playlist mutations (Rename, Delete, Create, Add/Remove Track).
- Automated test suites (Rust backend, React frontend unit tests, Playwright E2E).

## Known Limitations

- Waveform rendering is a static visual placeholder.
- Scrub controls are deferred.
- The agent currently stages metadata and playlist modifications but cannot automatically read/write to `master.db` natively; changes must be exported to XML and manually imported into Rekordbox.
